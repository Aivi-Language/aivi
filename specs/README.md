*   **[Syntax](02_syntax/)**: Detailed guide to language constructs.

*   **[Kernel](03_kernel/)**: The minimal core calculus underlying the language.
*   **[Desugaring](04_desugaring/)**: Formal translation from Syntax to Kernel.
*   **[Guides](guides/)**: Comparisons for developers coming from TS/Haskell.
*   **[Ideas](ideas/)**: Future directions and architectural possibilities.
