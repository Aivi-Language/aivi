# `@no_prelude` — Skip Prelude Import

<!-- quick-info: {"kind":"decorator","name":"@no_prelude"} -->
`@no_prelude` opts a module out of the implicit prelude import.
<!-- /quick-info -->

## Syntax

```aivi
@no_prelude module ModuleName
```
