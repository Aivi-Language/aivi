`@no_prelude` opts a module out of the implicit prelude import. Useful for low-level modules that intentionally avoid or redefine prelude symbols.
<!-- /quick-info -->

## Syntax

```aivi
@no_prelude module ModuleName
```

## Example

```aivi
// Opt out of the implicit prelude import.
// Useful for low-level modules that define their own primitives.
@no_prelude module Core.Primitives

// Nothing from the prelude is in scope here.
// You must import everything explicitly.
identity : a -> a
identity = x => x
```
